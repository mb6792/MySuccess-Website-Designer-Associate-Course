JavaScript Frameworks in 2020

It's 2020 and the JavaScript
train doesn't look like its about to slow down any time soon. As JavaScript evolves, the frameworks that go with it keep popping up and changing at at an amazing pace. Within three months, a framework can go from something nobody's ever heard of to the hottest new thing, only to be forgotten as soon as the next big thing comes along.

Let's look at some of this year's most popular JavaScript frameworks:

  1. Angular
  2. Backbone.js
  3. Ember
  4. React
  5. Vue

You should really take a look at the big five frameworks, and choose the one that best suits your needs. Personally, I'm using React quite a bit these days, as I find it provides just enough tools to get a project started without overwhelming me with built-in stuff. Here's how easy it is to create a simple React page: 

            ReactDOM.render(
                <h1<Hello, world!</h1>,
                document.getElementById('root')
